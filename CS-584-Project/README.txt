This project is written entirely in swift.

You would need MAC OS based Operating to run the project. We are not sure if this project can be run on Windows OS.

Setup: 

1. Install XCODE: https://developer.apple.com/xcode/resources/

2. Install OpenAISwift Library from github: https://github.com/adamrushy/OpenAISwift

3. Generate your API Key: https://platform.openai.com/account/api-keys
    (We cannot provide the api key because it's confidential information)

4. Paste your API key in ChatGPT/ChatGPT/ContentView.swift in line number 20.

5. Goto Product -> Destination -> and select your emulator type. (We developed it on Iphone 14).

5. You should be able to run the chat application on the emulator.
